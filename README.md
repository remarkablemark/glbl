# glbl

A module that is simply a reference to an [object](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Working_with_Objects).

## Install

```sh
$ npm install glbl
```

## Usage

```js
var obj = require('glbl');
obj.key = 'value';
console.log(obj); // { key: 'value' }
```

## Testing

```sh
$ npm test
$ npm run lint
```
